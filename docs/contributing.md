# Contributing (docs)

For the full contribution guide, see **[CONTRIBUTING.md](../CONTRIBUTING.md)** in the repository root.

Quick checklist before opening a PR:

- Run `uv run ruff check .` and `uv run ruff format --check .`
- Run `uv run mypy src/akc`
- Run `uv run pytest`
- Install pre-commit: `uv run pre-commit install`

Also see [CODE_OF_CONDUCT.md](../CODE_OF_CONDUCT.md) and [SECURITY.md](../SECURITY.md).
